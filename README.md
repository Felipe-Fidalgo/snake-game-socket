# 🐍 Jogo da Cobrinha Multiplayer (Snake Game)

Um dos primeiros projetos que desenvolvi no começo de meus aprendizados no desenvolvimento web. Com o tempo, decidi evoluir o projeto e adicionar novas funcionalidades, como suporte para um **segundo jogador** e a **possibilidade de jogar online** (multiplayer em tempo real).

## ✨ Funcionalidades

- Jogo clássico da cobrinha (Snake).
- Multiplayer (dois jogadores na mesma tela).
- Multiplayer online em tempo real (utilizando WebSockets com Socket.io).

## 💻 Como rodar o projeto localmente

Siga os passos abaixo para rodar o projeto na sua máquina:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/FelipeFidalgo19/Jogo_da_cobrinha.git
   ```

2. **Acesse o diretório do projeto:**
   ```bash
   cd Jogo_da_cobrinha
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```

4. **Inicie o servidor:**
   ```bash
   node server.js
   ```

5. **Acesse no navegador:**
   Abra o seu navegador e acesse `http://localhost:5000`
